# Motivationsschreiben_Generator

Ein KI-gestützter Generator, der automatisch professionelle Motivationsschreiben erstellt.

## Features

- Individuell zugeschnittene Motivationsschreiben basierend auf deinen Angaben
- Wahl zwischen Deutsch und Englisch
- Drei Schreibstile: Professionell & Formell, Freundlich & Modern, Selbstbewusst & Direkt
- Druckfertige Briefstruktur (Betreff, Anrede, Einleitung, Hauptteil, Schluss)
- Kopieren-Funktion für das generierte Schreiben

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript
- **Backend:** Node.js, Express
- **KI:** Google Gemini API (gemini-2.0-flash)
